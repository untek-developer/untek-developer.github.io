  • Response

Headers

  • Language: (string - default: ru)

    Язык

  • Time-Zone: (string - default: UTC)

    Часовой пояс

    Example:

    Asia/Almaty

Query Parameters

  • type: required (string)

    Тип уведомления (sms, email)

  • expand: (string)

    Включить в ответ дополнительные поля или связанные данные

    Example:

    profile,balance
  • fields: (string)

    Ограничение списка полей в теле ответа

    Example:

    id,title
  • per-page: (integer - default: 20 - minimum: 10 - maximum: 999999)

    Установить количество записей на странице

  • page: (integer - default: 1 - minimum: 1)

    С какой страницы начать выборку

  • sort: (string)

    Сортировка списка по полям. Для обратной сортировки нужно поставить знак минуса перед названием поля, например: -position

    Example:

    position,-date_change
  • search: (array)

    Поиск по указанным полям

    Example:

    search[name]=john
